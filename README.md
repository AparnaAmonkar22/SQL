# Monday Coffee Data Analysis (SQL Project)

An SQL-based data analysis project focused on extracting business insights from the Monday Coffee sales dataset.

## Project Overview
This project analyzes sales data for Monday Coffee to answer key business questions such as:
- Which products generate the most revenue?
- Which cities and stores perform best?
- How sales vary over time
- Decision-Making & Final Recommendations

The goal is to demonstrate practical SQL skills used in real-world data analysis.

## Tools & Technologies
- SQL (MySQL / PostgreSQL )
- DB  pgAdmin / MySQL Workbench
- CSV dataset

## Dataset Description
The dataset contains information related to:
- Orders
- Customers
- Products
- Store locations
- Sales and revenue

*(Dataset used for educational purposes.)*

## Key Questions Answered
- Total sales and revenue by product
- Monthly and daily sales trends
- Top-performing cities and stores
- Customer purchase behavior
- Average order value

## SQL Concepts Used
- `SELECT`, `WHERE`, `ORDER BY`
- `GROUP BY`, `HAVING`
- `JOIN` (INNER, LEFT)
- Subqueries
- Aggregate functions (`SUM`, `COUNT`, `AVG`)
- Date functions
